Editor Trees

The project entails writing methods for a class representing the underlying data structure of a text editor. Each piece of text is represented as a height-balanced tree with rank and balance codes, where each node contains a single character. This structure, known as an EditTree, focuses on the efficient manipulation of text rather than alphabetical ordering. The emphasis is on implementing complex data structures and associated algorithms, utilizing debugging tools effectively, interpreting algorithm descriptions for Java implementation, and practicing teamwork skills.

Database Project

The project aims to develop a ticket management system prioritizing real fans' access to their favorite musical artist's events and concerts. Unlike existing systems where prices fluctuate based on supply and demand, our system ensures fair access and affordability for fans while providing transparency to artists regarding profits. Additionally, the system may include features for fans to leave concert reviews. The primary objective is to deliver a functional system by the end of spring quarter 2024, enabling fans to access events and providing artists with valuable statistics about their fan base and events.

Arcade Game (Joust Advanced)

The project involves designing a game inspired by Joust, where a hero on an ostrich must defeat enemy knights riding buzzards by jousting with them. The game features multiple levels with different platform layouts and increasing difficulty. Players control the hero's movement using arrow keys, navigating platforms above a lava pool. Winning requires defeating all enemies while avoiding losing lives.
